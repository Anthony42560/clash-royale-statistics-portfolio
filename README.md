# Clash Royale Statistics Portfolio

This repository contains a statistical analysis of Clash Royale gameplay data collected over 30 days.

## Dataset

The dataset contains daily gameplay observations including:

- Hours Played
- Wins and Losses
- Mood Level (1–5)
- Trophy count
- Crown victories

Each row represents one day of gameplay.

## Methods Used

The analysis was performed using **R and RMarkdown** and includes:

- Data visualisation (ggplot2)
- t-test
- Linear regression
- Fisher's Exact Test
- Binomial regression

## Key Question

Does a player's mood significantly affect the probability of winning in Clash Royale matches after controlling for hours played?
